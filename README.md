# Open Co-curricular Practices
This repository hosts information on and files for open co-curricular practices (OCP) and open co-curricular resources (OCR).

## About

**What?** 
This repository features reusable resources in the form of OCP and OCR.

**Who?** 
The intended users are leaders/organizers of co-curricular activities, whether students or staff.

**Why?** 
The purpose of using these resources are to support inclusion, collaboration, and innovation in co-curricular activities. Users of these resources may also make their own adjustments that can be openly shared, which enables a cycle of constant improvement that benefits others. 

**How?** 
Resources on this repository are openly accessible. The expectation is that, should improvements be made, that they be openly shared in a similar manner.

For explanations of each resource, please view [the Wiki.](https://github.com/bj16xq/Open_Co-curricular_Practices/wiki)
